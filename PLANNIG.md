# QuestDo


## 🇬🇧 English Version

## 🧠 Idea

QuestDo is a gamified challenge platform where users complete real-life tasks (“quests”), earn points, and track progress with friends.

Focus:

* Motivation through gamification
* Real-world actions
* Simple social competition



## 🎯 Goal

Make everyday tasks more engaging by turning them into small challenges.

Examples:

* Sport & fitness
* Learning tasks
* Small daily challenges



## 🔁 Core Loop

1. Get or choose a quest
2. Complete it in real life
3. Mark as done
4. Earn points
5. Unlock next quests



## 🧩 Quests

* Created manually (admin / users)
* Users can submit ideas



## 🏆 Motivation

* Points per quest
* Levels
* Simple leaderboard
* Optional streaks



## 🌐 Platform

* Web app first
* Optional PWA later



## 🛠️ Tech

* Frontend: HTML / CSS / JavaScript
* Backend: Flask
* DB: SQLite



## 🚀 MVP

Must have:

* Users
* Quest list
* Complete quest button
* Points system + streaks
* Simple leaderboard



## 📈 Future

* Teams
* Weekly challenges

# 🇩🇪 Deutsche Version

## 🧠 Idee

QuestDo ist eine gamifizierte Challenge-Plattform, bei der Nutzer reale Aufgaben („Quests“) erledigen, Punkte sammeln und ihren Fortschritt mit Freunden verfolgen.

Fokus:

* Motivation durch Gamification
* Aktionen im echten Leben
* Einfache soziale Konkurrenz



## 🎯 Ziel

Alltägliche Aufgaben interessanter machen, indem sie in kleine Challenges verwandelt werden.

Beispiele:

* Sport & Fitness
* Lernaufgaben
* Kleine tägliche Herausforderungen



## 🔁 Kern-Loop

1. Quest erhalten oder auswählen
2. Im echten Leben erledigen
3. Als erledigt markieren
4. Punkte erhalten
5. Neue Quests freischalten



## 🧩 Quests

* Manuell erstellt (Admin / Nutzer)
* Nutzer können Ideen einreichen



## 🏆 Motivation

* Punkte pro Quest
* Level-System
* Einfache Rangliste (Leaderboard)
* Optionale Streaks



## 🌐 Plattform

* Zuerst Web-App
* Später optional als PWA



## 🛠️ Technik

* Frontend: HTML / CSS / JavaScript
* Backend: Flask
* Datenbank: SQLite



## 🚀 MVP

Muss enthalten:

* Nutzer
* Quest-Liste
* „Quest erledigt“-Button
* Punktesystem + Streaks
* Einfache Rangliste



## 📈 Zukunft

* Teams
* Wöchentliche Challenges
